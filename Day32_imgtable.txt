이미지가 웹에서 어떻게 작동하냐

1. src 안에다가 이미지 url 넣어준다
2. 이게 binary(숫자 덩어리)로 변한다

미리보기 작동방식
1.src 안에 base64라는 문자열 덩어리로 넣어준다
2.문자열 덩어리가 binary(숫자 덩어리)로 변한다


테이블 테니스 쿼리
CREATE TABLE t_board_imgs (
    id SERIAL PRIMARY KEY,
    imgurl VARCHAR NOT NULL DEFAULT '',
    original_filename VARCHAR NOT NULL DEFAULT '',
    unique_filename VARCHAR NOT NULL DEFAULT '',
    minetype VARCHAR NOT NULL DEFAULT '',
    filesize INT DEFAULT 0,
    board_id INT4 NOT NULL,
    created_dt TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    updated_dt TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    CONSTRAINT fk_board
        FOREIGN KEY (board_id)
        REFERENCES t_board (id)
        ON DELETE CASCADE -- 옵션: t_board 레코드 삭제 시 관련 이미지도 삭제
);
