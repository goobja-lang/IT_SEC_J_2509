400 (auth) : 권한이 없다. 너 저리 가셈
404(Not found) : 그런 API 주소(인터넷 주소)는 없다
505(Internal Server Error) : 서버에서 에러 터졌다

인터넷은 모든 데이터를 문자열로 주고 받아요.
서버도 문자열로 받고, 문자열로 줘요.
